# BORZO Script Navigator

Внутренний навигатор скриптов для менеджеров и руководителей BORZO.

## 🌐 Доступ

https://skvorcmen.github.io/

## 🔐 Доступ к системе

Доступ к системе осуществляется по ролям. Пароли для входа выдаются индивидуально.

## 📱 Возможности

### Для менеджеров:
- Просмотр всех скриптов (магистраль и ветки)
- Копирование текстов сообщений
- Предложение изменений существующих блоков
- Предложение новых блоков, элементов и разделов

### Для руководителей:
- Всё, что доступно менеджерам
- Редактирование любых блоков и сообщений
- Добавление и удаление блоков магистрали
- Добавление и удаление разделов и элементов
- Сортировка блоков перетаскиванием
- Экспорт базы скриптов
- Скачивание полного проекта

## 🔄 Сохранение сессии

После успешного входа сессия сохраняется в браузере на 24 часа.

## 🛠️ Технологии

- HTML5 / CSS3 / JavaScript
- LocalStorage для сохранения данных
- GitHub Pages для хостинга
